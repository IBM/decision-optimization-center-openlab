1. What is IBM Cloud ID?

IBM Cloud ID is the Identification-as-a-Service (IDaaS) from IBM. IBM Cloud ID is available for anyone who is using IBM cloud and many IBM cloud services.

2. Why am I getting a “No Resources Available” message?

This means that all of the current lab resources are being used by others. Please try again later.

3. What browsers are supported?

At this time, Chrome and Firefox browsers are supported.

4. What if I need more than 4 hours to complete a lab?

The initial time available to complete the lab is 4 (four) hours. However, should you need extra time to complete it, you can extend the lab up to 7 (seven) hours in 1 (one) hour increments.

5. What happens if I reach the maximum of 3 requests for a lab environment?

Users may send a Help request to extend the number of times to try the Lab.

6. What is the expected turnaround time if I open an issue?

There is no SLA for IBM Open Labs at this time; however, we typically respond to issues within 48 hours Monday-Friday based on priorities specified.
